# The Architecture of Autumn

I’ve always found the deep, quiet tones of autumn forests incredibly calming. This project started as a personal challenge to capture that exact mood and turn it into a clean, minimalist landing page. 

Instead of heavy scripts or canvas engines, I wanted to see how far I could push pure CSS and Tailwind CSS v4 to build something beautiful, lightweight, and fully responsive.

[Live Demo](https://mrkrub.github.io/autumn-cinematic-landing/) · [Explore Code](https://github.com/mrkrub/autumn-cinematic-landing)

---

## The Vibe & Mechanics

The goal was to make the page feel like a premium digital editorial magazine. Here is what I focused on to make it happen:

* **Real Glassmorphism:** I didn't want the blurry cards to look muddy. By layering soft borders (`border-white/10`) with a very light background tint and crisp `backdrop-blur-2xl`, the specular refraction looks incredibly clean over the dark forest colors.
* **Pure CSS Leaf Drift:** To keep performance flawless, the falling autumn leaves are powered entirely by GPU-accelerated CSS keyframes. No bulky JavaScript libraries, no jank—just smooth, lightweight motion.
* **Typographic Contrast:** I paired a sharp, modern sans-serif with an elegant, flowing italic serif to give the titles a sophisticated, high-end contrast.

---

## Tech Stack

* **HTML5** - Clean, semantic tags structured with accessibility (ARIA labels) in mind.
* **Tailwind CSS v4** - Used for rapid styling, taking full advantage of its next-gen cascading performance.
* **Google Fonts** - Playfair Display (for the organic editorial feel) & Inter for structural readability.
